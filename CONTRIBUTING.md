# Contributing

Thanks for your interest in contributing to `@andersseen/web-components`.

- Start with [`README.md`](./README.md) for project overview and setup.
- Read [`AGENTS.md`](./AGENTS.md) before making code changes — it contains the
  architecture rules, package boundaries, and testing requirements for this
  repository.
- Architecture details are in [`docs/SSD.md`](./docs/SSD.md), the task→file map
  is in [`docs/CODEMAP.md`](./docs/CODEMAP.md), and step-by-step recipes are in
  [`docs/AGENT-PLAYBOOKS.md`](./docs/AGENT-PLAYBOOKS.md).

Please open an issue to discuss large changes before submitting a pull request.
