✨ Key Features

  • Zero Runtime Dependencies: No Stencil, React, or Vue runtime

  • Headless Integration: Uses @andersseen/headless-components
  • TypeScript Native: Experimental decorators, full type safety
  • Shadow DOM: Style encapsulation, no CSS conflicts
  • Accessible: ARIA attributes, keyboard navigation
