# Andersseen Web Components

Welcome to the **Andersseen Web Components** Storybook! 🎉

These components are built with **Stencil**, making them framework-agnostic. You can use them in React, Angular, Vue, Astro, or plain HTML.

## Features

- 🧩 **Framework Agnostic**: Works perfectly with modern frameworks or vanilla JS.
- 🎨 **Tailwind CSS**: Designed seamlessly with Tailwind tokens.
- ♿️ **Accessible**: Built with accessibility in mind (WAI-ARIA).
- 🌓 **Dark Mode**: Fully supports light and dark themes using CSS Variables.

## Usage

You can find the components organized in the sidebar under `Components`. Each component includes:

- **Canvas**: Interactive sandbox to test different states.
- **Docs**: Comprehensive API documentation automatically generated from the source code.
- **Controls**: Tweak the props and CSS variables in real-time.

### Quick Start

\`\`\`html
<and-button variant="default" size="default">
Click Me
</and-button>
\`\`\`

## Documentation

Navigate to any component and click on the **Docs** tab at the top to view its documentation, props, and variants.
